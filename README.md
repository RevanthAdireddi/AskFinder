# AskFinder 2.0.0

AI-Powered Semantic File Search for Windows

## Overview

AskFinder is a desktop application that helps you quickly find files on your computer using natural language. Instead of searching only by filename, AskFinder understands the meaning of your query and searches your indexed documents intelligently.

## Features

* AI-powered semantic search
* Fast local file indexing
* Natural language search
* Search history
* Open files directly
* Open containing folders
* Modern desktop interface
* Completely offline after indexing

## System Requirements

* Windows 10 or Windows 11 (64-bit)
* At least 4 GB RAM (8 GB recommended)
* Approximately 250 MB of free disk space

## Installation

1. Run `AskFinder_2.0.0_Setup.exe`.
2. Follow the installation wizard.
3. Launch AskFinder from the Start Menu or Desktop shortcut.

## Getting Started

1. Open AskFinder.
2. Click **Build AI Index** to create your search index if searching is not successful.
3. Wait until indexing is complete.
4. Enter your search query.
5. Open the file or its containing folder directly from the search results.

## Updating the Index

Whenever you add, remove, or modify files, click **Update Index** to refresh the search database.

## Notes

* The first indexing process may take some time depending on the number of files.
* Search works best after the AI index has been built.
* All searching is performed locally on your computer.
