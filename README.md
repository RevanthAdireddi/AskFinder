Description:
AskFinder is an intelligent file search application that helps
users quickly find files on their computer using filename and
content-based searching.

# AskFinder

AI-powered semantic file search engine for Windows.

Website:
https://askfinder.kesug.com

Download:
https://github.com/RevanthAdireddi/AskFinder/releases


Features:
- Fast file searching
- Search by filename
- Search by file content
- Local database indexing
- Simple web-based interface
- One-click launch

How to Use:
1. Double-click launcher.exe
2. Wait for the application to start
3. Your default browser will open automatically
4. Enter your search query
5. View and open matching files

Requirements:
- Windows 10 or Windows 11
- Minimum 4 GB RAM
- No Python installation required

Project Information:
Project Name : AskFinder
Version      : 1.0
Developer    : Revanth Aditeddi

Notes:
- Keep all application files in the same folder.
- Do not delete database files.
- Use the Refresh Database option after adding new files.

Thank you for using AskFinder.
